# PSReactNativeDemo
PSReactNativeDemo集,里面有我的一些学习和研究的React Native第三方组件的Demo,希望可以帮到你!

## 使用
step1: git clone https://github.com/PengSiSi/PSReactNativeDemo.git
step2: 终端运行: npm install
step3: react-native run-ios
